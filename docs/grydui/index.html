Exemplo Rápido

1. Criar Container Application

npx create-gryd-app my-app create
# Escolha: Container Shell
# Microfrontends: users,products,orders

2. Criar Microfrontend

npx create-gryd-app users create
# Escolha: Microfrontend
# Port: 3001

3. Comunicação Entre Microfrontends

import { MicroFrontendEventBus } from '@gryd/api-client';

// Atualizar header do microfrontend
MicroFrontendEventBus.emit('page:title-changed', 'Users Management');
