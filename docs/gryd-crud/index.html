⚙️ Configuração

🔧 Registro de Serviços

Configure os serviços do GrydCrud Framework no Program.cs:

using GrydCrud.Application;
using GrydCrud.Infrastructure;

var builder = WebApplication.CreateBuilder(args);

// Registrar serviços do GrydCrud
builder.Services.AddApplicationGrydCrudServices();
builder.Services.AddInfrastructureGrydCrudServices(builder.Configuration);

var app = builder.Build();

📦 Instalação

Install-Package GrydCrud.Infrastructure
Install-Package GrydCrud.Application

Implementação Rápida

1. Entity Definition

public class Product : BaseEntity
{
    public string Name { get; set; }
    public decimal Price { get; set; }
    public string Description { get; set; }
}

2. Controller Implementation

[ApiController]
[Route("api/[controller]")]
public class ProductsController : BaseCrudController<Product, ProductDto>
{
    public ProductsController(IProductService service) : base(service) { }
}

3. React Hook Usage

const { data, loading, create, update, delete: remove } = useCrud<Product>('/api/products');

// Create new product
await create({ name: 'New Product', price: 99.99 });
